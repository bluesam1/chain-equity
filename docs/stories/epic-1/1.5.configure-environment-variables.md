### Relevant Source Tree Info

- Frontend `.env.local.example`: `frontend/.env.local.example`
- `.gitignore`: `chain-equity/.gitignore`
- README: `chain-equity/README.md`

### Architecture Decisions
- Environment variables stored in `.env` files (not committed to git)
- `.env.example` files serve as templates
- Vite requires `VITE_` prefix for environment variables exposed to frontend
- Backend uses standard environment variable names (no prefix)

### Key Constraints
- Never commit `.env` files with actual secrets
- Always provide `.env.example` files as templates
- Vite environment variables must use `VITE_` prefix
- Supabase credentials must be obtained from Supabase dashboard
