# ChainEquity PRD - 24 Hour Sprint

## Overview

This PRD is organized into 7 epics that can be worked through sequentially. Each epic contains all relevant requirements, deliverables, and implementation checklists in a single document.

## Epics

1. **[Epic 1: Project Foundation](./epic-1-foundation.md)**
   - Executive summary, project overview, technical stack, architecture decisions, and constraints

2. **[Epic 2: Core Smart Contract](./epic-2-smart-contract.md)**
   - ERC-20 token contract with allowlist, RBAC, virtual split, and mutable symbol

3. **[Epic 3: Backend Services](./epic-3-backend-services.md)**
   - Issuer service and cap-table generation service

4. **[Epic 4: Corporate Actions](./epic-4-corporate-actions.md)**
   - Stock split and symbol change implementation

5. **[Epic 5: Web UI](./epic-5-web-ui.md)**
   - Vite.js + React web interface with MetaMask integration and admin dashboard

6. **[Epic 6: Testing & Quality](./epic-6-testing-quality.md)**
   - Test scenarios, gas benchmarks, and success metrics

7. **[Epic 7: Documentation & Delivery](./epic-7-documentation-delivery.md)**
   - Submission requirements, timeline, final checklist, and reference materials

---

## Quick Navigation

- Start with [Epic 1: Project Foundation](./epic-1-foundation.md) for project setup and architecture
- Then proceed through epics 2-5 for implementation
- Use [Epic 6: Testing & Quality](./epic-6-testing-quality.md) for validation
- Complete with [Epic 7: Documentation & Delivery](./epic-7-documentation-delivery.md) for final submission
